# Hero背景图片设置指南

## 📸 背景图片要求

为了获得最佳的视觉效果，请按照以下要求准备背景图片：

### 🎯 图片内容建议
- **空气炸锅设备** - 现代、干净的空气炸锅产品图
- **美食场景** - 正在烹饪的食材或完成的菜品
- **厨房环境** - 现代厨房或烹饪场景
- **食材特写** - 新鲜的蔬菜、肉类、海鲜等
- **烹饪过程** - 食材在空气炸锅中的状态

### 📐 技术规格
- **分辨率**: 1920x1080 或更高 (推荐 2560x1440)
- **格式**: JPG 或 WebP
- **文件大小**: 小于 500KB (压缩后)
- **宽高比**: 16:9 或 21:9
- **质量**: 高质量，清晰锐利

### 🎨 视觉要求
- **光线**: 充足的自然光或柔和的人工光
- **色调**: 暖色调 (橙色、黄色、红色系)
- **对比度**: 中等对比度，不要太暗或太亮
- **构图**: 留出文字显示空间，避免过于复杂的背景

## 🚀 安装步骤

1. **准备图片**
   - 按照上述要求拍摄或选择合适的图片
   - 使用图片编辑软件调整尺寸和压缩

2. **命名文件**
   - 将图片命名为 `hero-background.jpg`
   - 确保文件名完全匹配

3. **放置文件**
   - 将图片放在 `public/images/` 目录下
   - 完整路径: `public/images/hero-background.jpg`

4. **验证效果**
   - 启动开发服务器: `npm run dev`
   - 访问首页查看效果
   - 调整图片透明度或替换图片

## 🎭 透明度调整

如果需要调整背景图片的透明度，编辑 `src/components/Hero.tsx` 文件中的渐变遮罩层：

```tsx
{/* 渐变遮罩层 */}
<div className="absolute inset-0 bg-gradient-to-br from-orange-900/40 via-orange-800/30 to-orange-900/40"></div>
<div className="absolute inset-0 bg-gradient-to-t from-black/20 via-transparent to-black/10"></div>
```

调整 `/40`、`/30`、`/20`、`/10` 等数值来改变透明度。

## 📱 响应式优化

背景图片已经优化为响应式：
- 使用 `object-cover` 确保图片覆盖整个区域
- 使用 `fill` 和 `sizes="100vw"` 确保最佳性能
- 移动端和桌面端都有良好的显示效果

## 🔧 故障排除

### 图片不显示
1. 检查文件路径是否正确
2. 确认文件名拼写无误
3. 检查文件权限

### 图片质量不佳
1. 使用更高分辨率的原图
2. 调整压缩设置
3. 考虑使用 WebP 格式

### 文字可读性差
1. 调整渐变遮罩层的透明度
2. 选择对比度更高的背景图片
3. 调整文字阴影效果

## 💡 设计建议

1. **主题一致性**: 确保背景图片与网站主题一致
2. **品牌色彩**: 考虑使用与品牌色彩协调的图片
3. **季节更新**: 可以根据季节或节日更换背景图片
4. **A/B测试**: 尝试不同的背景图片来优化转化率

---

**注意**: 如果暂时没有合适的背景图片，网站会显示默认的渐变背景，不影响正常使用。
