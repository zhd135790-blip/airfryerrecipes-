# Contributing to Air Fryer Recipes

Thank you for your interest in contributing to Air Fryer Recipes! We welcome contributions from the community to help make this project better.

## üöÄ How to Contribute

### 1. Fork the Repository

- Click the "Fork" button on the GitHub repository page
- Clone your fork locally: `git clone https://github.com/yourusername/air-fryer-recipes.git`

### 2. Create a Branch

```bash
git checkout -b feature/your-feature-name
# or
git checkout -b fix/your-bug-fix
```

### 3. Make Your Changes

- Follow the existing code style
- Test your changes thoroughly
- Update documentation if needed

### 4. Commit Your Changes

```bash
git add .
git commit -m "Add: your descriptive commit message"
```

### 5. Push and Create a Pull Request

```bash
git push origin feature/your-feature-name
```

Then create a Pull Request on GitHub.

## üç≥ Recipe Contribution Guidelines

### Recipe Requirements

- **Tested**: All recipes must be tested in a real air fryer
- **Precise**: Include exact cooking times and temperatures
- **Complete**: Provide ingredients, instructions, and nutrition info
- **Quality**: Use high-quality, appetizing photos

### Recipe Format

Follow the existing Markdown format in `content/recipes/`:

```markdown

excerpt: Brief description

tags: [tag1, tag2, tag3]

  - 1 cup ingredient
  - 2 tbsp seasoning
```

## üìã Pull Request Guidelines

### Before Submitting

- [ ] Test your changes locally
- [ ] Ensure all tests pass
- [ ] Update documentation if needed
- [ ] Follow the existing code style
- [ ] Add appropriate commit messages

### PR Description

Include:

- **What**: Description of changes
- **Why**: Reason for the changes
- **How**: How the changes work
- **Testing**: How you tested the changes

### Review Process

- All PRs require review before merging
- Address feedback promptly
- Keep PRs focused and small when possible
- Update your PR if requested

## üêõ Reporting Issues

### Bug Reports

When reporting bugs, please include:

- **Description**: Clear description of the issue
- **Steps to Reproduce**: How to reproduce the bug
- **Expected Behavior**: What should happen
- **Actual Behavior**: What actually happens
- **Environment**: Browser, OS, device info
- **Screenshots**: If applicable

### Feature Requests

For new features:

- **Description**: What feature you'd like
- **Use Case**: Why this feature would be useful
- **Proposed Solution**: How you think it should work
- **Alternatives**: Other ways to solve the problem
